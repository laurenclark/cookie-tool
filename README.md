# pecr-cookie-optin

Uses:-
- Parcel
- Babel Core and Preset Env
- PostCSS SugarSS for Sass syntax 
- PostCSS Nested
- PostCSS Modules
- Autoprefixer

`yarn start`

Runs parcel on localhost:1234

`yarn build`

Builds to dist folder


If using VSCode add this to your settings to get prettier to work with .sss files
```
  "files.associations": {
      "*.css": "postcss",
      "*.sss": "sass"
  },
```
